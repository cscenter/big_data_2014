Домашнее задание №7
Шашкова Елизавета, CSC, 2014

Поиск схожих текстов при помощи построения разреженного кольчужного покрытия.

Входом программы является каталог с файлами, где каждый файл
– текстовый документ из корпуса, а также булевский флажок -s, сигнализирующий, нужно ли строить
разреженное покрытие или плотное.

Важно! Целочисленный аргумент n, означающий размер звена, нужно менять в файле shingling.py вручную.

Пример запуска программы: 
python jaccard.py -s data

(Затем дважды запустить mincemeat)
